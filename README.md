## Quick Start (First Time)

### Step 1 — Install Python dependencies

```bash
pip3 install -r requirements.txt
# or if needed:
pip3 install flask PyJWT --break-system-packages
```

### Step 2 — Start the server

```bash
chmod +x start.sh
./start.sh
```

Or manually:

```bash
cd backend
python3 app.py
```

### Step 3 — Open in browser

```
http://localhost:5001
```

That's it. Flask serves both the API and the React frontend from a single port.

## Admin Access

| URL | Credentials |
|---|---|
| http://localhost:5001 | Click "Admin" in the footer |
| Direct: http://localhost:5001 → navigate to Admin | admin / admin123 |

**Change the default password** via the admin dashboard after first login, or set environment variables:

```bash
ADMIN_USERNAME=myuser ADMIN_PASSWORD=mysecretpw python3 app.py
```

## Rebuilding the Frontend

If you modify any file inside `frontend/src/`, rebuild:

```bash
chmod +x build.sh
./build.sh
```

Then restart the Flask server.

### Requirements for rebuilding

- Node.js installed
- `tsx` installed globally: `npm install -g tsx`
- `react` and `react-dom` installed globally: `npm install -g react react-dom`

## API Reference

All endpoints return JSON.

### Public

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/properties` | List published properties (supports filters: q, country, property_type, bedrooms, status, max_rent) |
| GET | `/api/properties/:id` | Get property by ID |
| GET | `/api/properties/slug/:slug` | Get property by slug |
| POST | `/api/inquiries` | Submit a rental inquiry |

### Auth

| Method | Endpoint | Description |
|---|---|---|
| POST | `/api/auth/login` | Login → returns JWT token |
| GET | `/api/auth/verify` | Verify token is valid |

### Admin (requires Bearer token)

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/admin/properties` | List ALL properties (including drafts) |
| POST | `/api/admin/properties` | Create property |
| PUT | `/api/admin/properties/:id` | Update property |
| DELETE | `/api/admin/properties/:id` | Delete property |
| PATCH | `/api/admin/properties/:id/publish` | Toggle published/draft |
| GET | `/api/admin/inquiries` | List all inquiries |
| PATCH | `/api/admin/inquiries/:id` | Update inquiry status |
| GET | `/api/admin/stats` | Dashboard statistics |
| POST | `/api/admin/change-password` | Change admin password |

## Database

SQLite database stored at `backend/erowho.db`. Created automatically on first run.

Tables:
- `properties` — all rental property listings
- `inquiries` — rental and contact form submissions
- `admins` — admin accounts (hashed passwords)

To reset the database:
```bash
rm backend/erowho.db
python3 backend/app.py  # recreates and re-seeds
```

## Production Deployment Notes

1. Set a strong `SECRET_KEY` environment variable
2. Set a strong `ADMIN_PASSWORD`
3. Use `gunicorn` instead of Flask's dev server:
   ```bash
   pip install gunicorn
   gunicorn -w 4 -b 0.0.0.0:5001 backend.app:app
   ```
4. Put Nginx in front for SSL termination
5. Replace SQLite with PostgreSQL for higher traffic
